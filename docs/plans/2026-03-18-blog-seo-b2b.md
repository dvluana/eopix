# Blog SEO + B2B Landing Pages Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Implementar blog com Sanity CMS, páginas B2B segmentadas, sitemap dinâmico e geração de imagens com IA — construindo funil SEO do zero para capturar MEI e empresas B2B (contabilidades, imobiliárias, varejistas).

**Architecture:** Sanity CMS como source of truth de conteúdo → `next-sanity` GROQ para fetching no App Router → páginas com ISR (revalidate 60s) + SSG para posts estáticos. Studio embedado em `/studio` (acessível só internamente). B2B landing pages são estáticas (sem CMS). Geração de imagens é script local offline (não corre no runtime do Next.js).

**Tech Stack:** next-sanity v9+, @sanity/image-url, @google/generative-ai (Imagen), Next.js 14 App Router, Zod, Vitest.

```typescript
import { groq } from 'next-sanity'

// Lista de posts para /blog (apenas campos de preview)
export const allPostsQuery = groq`
  *[_type == "blogPost" && defined(publishedAt)] | order(publishedAt desc) {
    _id,
    title,
    slug,
    publishedAt,
    excerpt,
    segment,
    mainImage
  }
`

// Posts recentes para home (últimos 3)
export const recentPostsQuery = groq`
  *[_type == "blogPost" && defined(publishedAt)] | order(publishedAt desc) [0..2] {
    _id,
    title,
    slug,
    publishedAt,
    excerpt,
    segment
  }
`

// Post individual por slug
export const postBySlugQuery = groq`
  *[_type == "blogPost" && slug.current == $slug][0] {
    _id,
    _createdAt,
    title,
    slug,
    publishedAt,
    excerpt,
    primaryKeyword,
    segment,
    body,
    mainImage,
    estimatedReadingTime
  }

// Todos os slugs (para generateStaticParams)
```

```tsx
export async function generateMetadata({ params }: PageProps): Promise<Metadata> {
  const { slug } = await params
  const post: BlogPost | null = await client.fetch(postBySlugQuery, { slug })
  if (!post) return {}

    title: post.title,
    description: post.excerpt,
    keywords: [post.primaryKeyword],

      description: post.excerpt,
      type: 'article',
      publishedTime: post.publishedAt,
      url: `https://somoseopix.com.br/blog/${slug}`,
    },
  }
}
```

```typescript
  const outputDir = 'public/blog'
  if (!fs.existsSync(outputDir)) {
    fs.mkdirSync(outputDir, { recursive: true })
  }

  const slug = postTitle.toLowerCase()
    .normalize('NFD').replace(/[\u0300-\u036f]/g, '')
    .replace(/[^a-z0-9]+/g, '-')
    .replace(/^-+|-+$/g, '')

  const promptFile = path.join(outputDir, `${slug}-prompt.txt`)
  fs.writeFileSync(promptFile, prompt)
  console.log(`\nPrompt salvo em: ${promptFile}`)
}

const [, , title, context] = process.argv
if (!title) {
  console.error('Uso: npx tsx scripts/generate-blog-image.ts "Título do post" "contexto opcional"')
  process.exit(1)
}

generateBlogImage(title, context)
```
